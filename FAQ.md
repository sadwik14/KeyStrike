# Frequently Asked Questions

## General

**Q: What is KeyStrike?**  
A: A professional typing speed test platform with real-time WPM tracking and global leaderboards.

**Q: Is it free?**  
A: Yes, completely free forever. No signup required.

**Q: Do I need to create an account?**  
A: No, results are saved in your browser automatically.

**Q: How is WPM calculated?**  
A: WPM = (characters typed / 5) / (time in minutes). Standard industry formula.

**Q: What's a good typing speed?**  
A: Average is 40 WPM. Good is 60-80 WPM. Excellent is 100+ WPM.

## Technical

**Q: Where is my data stored?**  
A: In your browser's localStorage by default. Optionally in MongoDB for cloud sync.

**Q: Can I use it offline?**  
A: Yes, once loaded. Results save when you're back online.

**Q: Which browsers are supported?**  
A: Chrome, Firefox, Safari, Edge (latest versions).

**Q: Is it mobile-friendly?**  
A: Yes, fully responsive on all devices.

**Q: Can I export my results?**  
A: Coming in v1.1. Currently view in History page.

## Features

**Q: How does the streak work?**  
A: Counts consecutive days with at least 1 test. Resets if you miss a day.

**Q: Can I customize test duration?**  
A: Yes, choose 15s, 30s, or 60s modes.

**Q: Can I practice with custom text?**  
A: Coming in v1.1. Currently uses curated texts.

**Q: Are there typing tips?**  
A: Yes, visit the Tips page for comprehensive guides.
